= rokol/examples

Run example with Cargo:

[source,sh]
----
$ cargo run --example 2_window
----

Run example with Cargo, forcing OepnGL renderer:

[source,sh]
----
$ env ROKOL_RENDERER=GlCore33 cargo run --example 5_quad
----

== Supported shaders

|===
| example    | GLSL (GlCore33) | Metal | HLSL (D3D11)

| 1_window   | yes             | yes   |
| 2_clear    | yes             | yes   |
| 3_triangle | yes             | yes   |
| 4_quad     | yes             | yes   |
| 5_texture  | yes             | yes   |
|===
